# 📊 Dashboard de Performance de Corridas — Power BI

![Dashboard](images/capa.png)

---

## 📌 Sobre o Projeto
Este projeto foi desenvolvido com o objetivo de analisar o desempenho em corridas utilizando dados do aplicativo Adidas Running.

A proposta é transformar dados brutos em insights visuais que permitam acompanhar evolução, consistência e performance ao longo do tempo, utilizando visualizações interativas e indicadores de desempenho.

---

## 🎯 Objetivo
- Monitorar evolução de desempenho
- Identificar padrões nos treinos
- Comparar métricas ao longo do tempo

---

## 🛠️ Ferramentas utilizadas
- Power BI
- DAX
- Excel

---

## 📈 Métricas analisadas
- Distância total percorrida
- Ritmo médio (pace)
- Velocidade máxima
- Calorias totais queimadas

## 🔍 Principais análises
- Evolução da performance ao longo dos meses
- Comparação de ritmo médio entre períodos
- Identificação dos melhores desempenhos individuais (ranking de corridas)
- Análise detalhada por treino com uso de tooltips interativos

---

## 🚀 Insights gerados
- Identificação de padrões de melhoria no ritmo
- Destaque para treinos com melhor desempenho individual
- Relação entre volume de corrida e calorias queimadas
- Análise de consistência e evolução ao longo do tempo

---

## 🧠 Diferenciais do dashboard
- Uso de tooltips personalizados para análise detalhada por treino
- Navegação interativa entre visão geral e análise detalhada
- Design focado em usabilidade e leitura rápida de indicadores
